# Guide Utilisateur CZI

## 1. Demarrage

1. Creer un compte via `Signup`.
2. Se connecter via `Login`.
3. Completer l onboarding en 6 etapes.
4. Attendre la validation admin si necessaire.

## 2. Utiliser la plateforme

### 2.1 Profil membre

- Aller sur `Membres` pour consulter les profils visibles.
- Ouvrir une fiche membre pour consulter les informations.
- Utiliser `Contacter` / `Appeler` si disponible.

### 2.2 Communaute

- Aller sur `Communaute`.
- Choisir une communaute/sous-communaute visible.
- Publier un message.
- Repondre, liker, modifier son propre message.

### 2.3 Messagerie privee

- Creer une discussion directe avec un autre membre.
- Lire les messages et repondre.
- Verifier les badges non lus.

### 2.4 Communiques

- Aller sur `Communiques`.
- Lire les annonces qui correspondent a votre scope.

### 2.5 Support CZI

- Aller sur `Support`.
- Poser une question simple.
- Lire la reponse dans le panneau principal.

## 3. Bonnes pratiques

- Utiliser des messages courts et clairs.
- Verifier les informations avant envoi.
- Signaler tout probleme a l admin.

## 4. En cas de probleme

- Deconnexion / reconnexion.
- Verifier la connexion internet.
- Si le probleme persiste, contacter l admin CZI.
